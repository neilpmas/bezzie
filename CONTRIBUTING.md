# Contributing

## Setup

```sh
git clone https://github.com/neilpmas/bezzie.git
cd bezzie
npm install
```

## Development

```sh
npm test        # run tests
npm run lint    # lint
npm run build   # compile to dist/
```

## Submitting a PR

1. Fork the repo
2. Create a branch: `git checkout -b my-fix`
3. Make your changes — tests must pass and lint must be clean
4. Open a pull request against `main`
